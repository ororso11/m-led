## 시스템 개요

### 기술 스택
- **프론트엔드**: HTML, CSS, JavaScript
- **백엔드**: Firebase (Realtime Database, Storage, Authentication)
- **호스팅**: Vercel

### 주요 기능
- 제품 카탈로그 목록 및 상세 보기
- 필터 및 검색 기능
- 관리자 페이지를 통한 제품 관리
- 실시간 데이터 동기화

## 관리자 로그인

### 1. 로그인 페이지 접속
```
https://your-domain.vercel.app/login.html
```

### 2. 관리자 계정 정보
- **이메일**: Firebase Console에서 등록한 이메일 : admin@intechlighting.com
- **비밀번호**: 설정한 비밀번호 : admin1234

### 3. 로그인 프로세스
1. login.html 페이지 접속
2. 이메일과 비밀번호 입력
3. "로그인" 버튼 클릭
4. 자동으로 관리자 페이지(admin.html)로 이동

### 4. 로그아웃
- 관리자 페이지 우측 상단 "로그아웃" 버튼 클릭

## 제품 등록

### 1. 관리자 페이지 접속
```
https://your-domain.vercel.app/admin.html
```

### 2. 제품 추가 탭에서 정보 입력

#### 필수 정보
- **제품명**: 제품의 정식 명칭
- **제품번호**: 제품 고유번호 (선택사항)
- **썸네일 이미지**: 목록에 표시될 대표 이미지 (1장)
- **상세 이미지**: 상세 페이지에 표시될 이미지 (여러 장 가능)

#### 제품 사양 (줄바꿈으로 구분)
```
예시:
150-400mA|resistor|
28000673
```

#### 제품 상세 스펙
- "스펙 항목 입력" 필드에 한 줄씩 입력 후 "추가" 버튼 클릭
- 예: `CRI > 90`, `색온도: 3000K` 등

#### 제품 테이블 데이터
- 품목, 전압, 전류, 최대출력, 효율, 크기, 보증기간 입력
- 입력하지 않은 항목은 자동으로 `-`로 표시됨

#### 카테고리 설정 (필터용)
- **WATT**: 0-5W, 6-10W, 11-15W, 16-20W, 21-25W, 26-30W, 30W+
- **CCT**: 2400K, 2700K, 3000K, 3500K, 4000K, 5700K, 6000K, 6500K, TW, RGB, RGBW
- **IP**: IP20, IP44, IP54, IP65, IP66, IP67, IP68

### 3. 제품 추가 버튼 클릭
- 모든 정보 입력 후 하단의 "제품 추가" 버튼 클릭
- 업로드 완료 시 "제품이 추가되었습니다! (즉시 반영)" 메시지 확인
- 자동으로 메인 사이트에 실시간 반영됨

## 제품 수정 및 삭제

### 제품 수정
1. 관리자 페이지 → "제품 관리" 탭 클릭
2. 수정할 제품의 "수정" 버튼 클릭
3. 자동으로 "제품 추가" 탭으로 이동하며 기존 데이터가 채워짐
4. 수정할 내용 변경
   - 이미지를 변경하지 않으려면 파일을 선택하지 마세요
   - 이미지를 변경하려면 새 파일 선택
5. "제품 수정 완료" 버튼 클릭

### 제품 삭제
1. 관리자 페이지 → "제품 관리" 탭 클릭
2. 삭제할 제품의 "삭제" 버튼 클릭
3. 확인 메시지에서 "확인" 클릭
4. 즉시 삭제되며 메인 사이트에 실시간 반영됨

## 사이트 구조

### 메인 페이지 (index.html)
- 제품 카탈로그 목록
- 좌측 사이드바: 필터 (WATT, CCT, IP)
- 상단 검색창: 제품명/사양 검색
- 페이지네이션: 페이지당 10개 제품 표시

### 제품 상세 페이지
- 제품 클릭 시 자동으로 상세 페이지 표시
- 상세 이미지, 스펙 리스트, 테이블 정보 표시
- "✕" 버튼으로 목록으로 돌아가기

### 관리자 페이지 (admin.html)
- **제품 추가 탭**: 새 제품 등록
- **제품 목록 탭**: 간단한 목록 보기
- **제품 관리 탭**: 수정/삭제 가능한 카드 형식 목록

## 주의사항

### 이미지 업로드
- **권장 이미지 크기**: 
  - 썸네일: 800x800px 이하
  - 상세 이미지: 1200px 이하
- **파일 형식**: JPG, PNG
- **파일명**: 한글 포함 시 자동으로 변환됨
- **용량**: 개당 2MB 이하 권장

### Firebase Storage 용량
- **무료 할당량**: 5GB 저장, 1GB/일 다운로드
- **현재 사용량 확인**: Firebase Console → Storage

### 데이터 동기화
- 제품 추가/수정/삭제는 **즉시 메인 사이트에 반영**됨
- 브라우저 새로고침 불필요
- 여러 관리자가 동시 작업 가능 (실시간 동기화)

### 보안
- 관리자 페이지는 로그인 필수
- 로그인 없이 admin.html 직접 접속 시도 시 login.html로 리다이렉트
- 관리자 계정 정보는 절대 공유하지 마세요

## 문제 해결

### 로그인이 안 될 때
1. 이메일/비밀번호 재확인
2. 브라우저 캐시 삭제 후 재시도
3. Firebase Console → Authentication → Users에서 계정 존재 확인

### 이미지가 업로드되지 않을 때
1. 파일 형식 확인 (JPG, PNG만 가능)
2. 파일 크기 확인 (5MB 이하)
3. 인터넷 연결 상태 확인
4. 브라우저 콘솔(F12)에서 에러 메시지 확인

### 제품이 메인 사이트에 표시되지 않을 때
1. 관리자 페이지 "제품 목록" 탭에서 제품 존재 확인
2. 메인 사이트에서 브라우저 새로고침 (Ctrl+F5)
3. 카테고리 필터가 적용되어 있는지 확인
4. Firebase Console → Realtime Database에서 데이터 확인

### 필터가 작동하지 않을 때
1. 제품 등록 시 카테고리를 정확히 설정했는지 확인
2. 브라우저 새로고침 (Ctrl+F5)
3. 브라우저 콘솔(F12)에서 에러 확인

## Firebase Console 접속

제품 데이터 직접 확인/수정이 필요한 경우:

1. https://console.firebase.google.com 접속
2. catalog-led 프로젝트 선택
3. **Realtime Database**: 제품 데이터 확인
4. **Storage**: 업로드된 이미지 확인
5. **Authentication**: 관리자 계정 관리
